O que são módulos para arduino ?

Módulos Arduino são dispositivos eletrônicos formados a partir de placas de circuitos impresso e componentes específicos, a depender da finalidade para o qual foram desenvolvidos, variando também em tamanho, cor, valores de venda, etc. Em resumo, um módulo para arduino é a forma como são conhecidos diversos componentes integrados em uma única placa com uma finalidade específica, aumentando as funcionalides do microcontrolador principal.

Qual a função do módulo arduino ?

A função do Módulo Arduino depende de quais componentes foram utilizados em sua produção, por exemplo, um módulo relé é um conjunto completo com demais componentes conectados para controles de carga, já um módulo dimmer é um dispositivo especifico para dimerizar lâmpadas incandescentes possibilitando o controle preciso da iluminação. A função depende da finalidade para a qual foi criado.

Qual a diferença entre os módulos arduino ?

A diferença entre os módulos Arduino está na finalidade para o qual foram criados, apresentando funções, componentes e objetivos diferenciados. São um conjunto muito extenso de acessórios para microcontroladores que foram implementados ao longo do tempo.
